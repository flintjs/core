# Stopwatch

`Stopwatch` is a lightweight utility class for tracking elapsed time with high precision using Node.js `performance.now()`. It supports starting, stopping, restarting, and formatting durations with optional decimal precision.

## Constructor

```ts
new Stopwatch(digits?: number)
````

* `digits` (optional): Number of decimal places to display when converting duration to string. Defaults to `2`.

### Example

```ts
const timer = new Stopwatch()
```

## Properties

### `duration: number`

Returns the elapsed time in milliseconds since the stopwatch started. If the stopwatch has been stopped, it returns the duration between start and stop.

```ts
const timer = new Stopwatch()
// ...some code
console.log(timer.duration) // e.g., 123.45
```

### `running: boolean`

Indicates whether the stopwatch is currently running.

```ts
timer.running // true or false
```

### `digits?: number`

Number of decimal places to use when formatting the duration in `toString()`. Defaults to `2`.

---

## Methods

### `restart(): this`

Resets the start time to the current time and sets the stopwatch as running.

```ts
timer.restart()
```

### `reset(): this`

Stops the stopwatch and resets both start and end times to the current time.

```ts
timer.reset()
```

### `start(): this`

Starts the stopwatch if it is not running. If it was previously stopped, it resumes from the previous elapsed duration.

```ts
timer.start()
```

### `stop(): this`

Stops the stopwatch, preserving the current duration.

```ts
timer.stop()
```

### `toString(): string`

Returns a formatted string representing the elapsed duration with automatic unit selection:

* `μs` for microseconds (< 1ms)
* `ms` for milliseconds (1ms ≤ duration < 1000ms)
* `s` for seconds (≥ 1000ms)

```ts
console.log(timer.toString()) // "123.45ms"
```

---

## Example Usage

```ts
const timer = new Stopwatch(3) // 3 decimal places
// ...some operation
console.log(`Operation took ${timer.stop().toString()}`)
```

This will output a human-readable string like:

```
Operation took 1.234s
```
