# Welcome to Flint

Flint is a **TypeScript-first** bot framework built on top of [`@fluxerjs/core`](https://fluxer.js.org) that gives you a clean, opinionated way to build Fluxer bots.
Instead of wiring everything up yourself, Flint handles the boilerplate, command handling, event listening, preconditions and more, so you can focus on building the things that actually matter.

```ts
export default defineCommand({
    name: "ping",
    description: "Pong!",

    async execute(client, message) {
        await message.reply("Pong! 🏓")
    }
})
```
